# DESCANSO System

DESCANSO is not:

* a music release
* a meditation product
* an immersive experience
* a streaming platform

It is:

* a listening territory
* a spatial work
* an ongoing practice of attention and suspension

The subsystem should reduce:

* stimulation
* speed
* informational density

And increase:

* duration
* silence
* presence
* space

The visitor should feel:
less stimulus while navigating.

Avoid:

* large navigation systems
* excessive explanation
* strong contrast
* heavy interaction
* aggressive visual hierarchy

DESCANSO pages should feel slower than the main website.
